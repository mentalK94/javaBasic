## Chap05-OpenChallenge
- Bear's Fish eating game

### 요구사항
- 게임에는 Bear와 Fish가 등장하며 10행 20열의 격자판에서 각각 정해진 규칙에 의해 움직인다.
- Bear는 사용자의 키에 의해 왼쪽(a키), 아래(s키), 위(d키), 오른쪽(f키)으로 한 칸씩 움직인다.
- Fish는 다섯 번 중 세 번은 제자리에 있고, 나머지 두 번은 4가지 방향 중 랜덤하게 한 칸씩 움직인다.
- 게임은 Bear가 Fish를 먹으면 성공으로 끝난다.
- 객체의 이동을 정의하는 move(), 모양을 정의하는 getShape()을 추상 메소드로 가진 추상 클래스 GameObject이다.
- GameObject를 이용해 Bear와 Fish클래스를 작성해라.
- MVC패턴을 사용해라.
- (추가사항) GUI로 구현해라
- (추가사항) Bear와 Fish의 모양을 이미지로 나타내어라.
- (추가사항) Bear와 Fish의 모양을 Custom 이미지로 변경할 수 있도록 기능을 추가해라.
- (추가사항) 커스텀 키 기능을 추가해라.
